Master Page Layout

These following are proposed design layouts for use on web applications maintained by the Environmental Monitoring, Reporting, and Economics Unit, Knowledge Management Branch, Ministry of Enviroment.

The layouts are using the Bootstrap open source framework. This framework is used by the B.C. government and recommended to meet citizen expectations for mobile responsive design and other web based soltions. In choosing to leverage Bootstrap, the B.C. government is also committing to Bootstrap`s supported browser model for application development standards.

NOTE: Not all Bootstrap layout and design elements below have been approved for use within the B.C. government corporate online style implementation. They are displayed here for prototyping purposes and to create meaningful dialogue on further enhancing the citizen user experience.

Messages

Success Message

Used when the selected action has completed successfully.

Your request was completed successfully.

Your requested actions are complete and you can continue doing some other thing.

If required, put some additional information here.

Next Steps: you can create a PDF version of your receipt or send a copy by email.

Information Message

Used when the user should be made aware of something, but they can proceed if they ignore the message.

Here is some info you could use.

When you select more than one payment type, the system will create two different payment lists for you.

Note: if you don't want to pay online, you can pay in person at participating BC Government agent offices.

Warning Message

Used when the user has a form-level validation error or similar. They can't proceed unless they resolve the issue.

We were not able to complete the registration request.

Reason: The information provided does not match an account in our system.

You may update the information and try again, or you can search for and pay invoices inividually through ENV even if you don't register.

Note: after three unsuccessful registration attempts you will not be able to make further registration attempts online.

Error Message

Used when the system has encountered an internal error or the user is unable to proceed with the chosen action.

Your request could not be completed due to an error.

Reason: One of the external data systems we connect to is not available.

If this is the first time you have received this message, please try again in a few minutes. If you continue to receive this message please contact the ENV Support Desk for assistance.

Note: your data has not been saved and no payments or transactions have been completed. We appologize for the inconvenience.
